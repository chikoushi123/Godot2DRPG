# 2D Platformer Prototype (Godot Engine)

This project is a small 2D platformer prototype developed using the **Godot Engine**.  
The goal of the project is to demonstrate core gameplay mechanics, basic AI behaviour, and interactive systems within a simple pixel-art environment.

The game focuses on movement, platform navigation, and interaction with objects such as coins, enemies, and hazards.

---

## 🎮 Gameplay Overview

The player controls a character that can move and jump across a tile-based platform world.  
Coins are scattered throughout the level and can be collected to increase the player's score.

Enemies patrol certain areas of the map, creating obstacles that the player must avoid.  
The level design encourages timing, positioning, and exploration.

## ⚙ Implemented Systems

### Player Movement
The player can move left and right and jump between platforms using simple physics-based movement.

### Coin Collection System
Coins can be collected by the player, increasing the score counter displayed on the screen.

### Enemy Behaviour
Enemies move back and forth across platforms using raycasts to detect edges and obstacles.

### Score System
A basic score system updates the UI whenever the player collects coins.

### Tilemap Level Design
The environment is built using a tilemap system to create modular platform structures.

### Background Music and Sound Effects
The project includes background music and sound effects for actions such as jumping and collecting coins.
